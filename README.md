# AnyNote
**AnyNote** 是一款極簡的便利貼應用程式
，以 Python + PyQt5 開發，可支援多個便利貼、字體切換、顏色樣板與自動儲存功能。  
目前提供 Windows 桌面版下載。

---

---

## ✨ 功能特色
- 🗒️ 多便利貼視窗，可自由拖曳與關閉  
- 💾 自動儲存內容，重開程式自動載入  
- 🎨 支援 4 種字體與多色便利貼樣式  
- 🔤 字體顏色可切換  
- 📌 支援 JSON 格式儲存多筆筆記
